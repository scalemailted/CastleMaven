Castle Maven

Gothic castle crawl — HTML + CSS + minimal JS

A minimalist interactive-story game. Each room is a plain HTML page with an image map—hover or click hotspots to explore, solve a small puzzle in the library, avoid a few nasty ends, and (maybe!) unlock the treasure vault. No backend, no build step—just static files.

Play Game View Code
How to play:
  • Hover or click hotspots on each scene image; the cursor and tooltip hint what you’ll find.
  • Garden & Kitchen: lateral exploration; the kitchen includes ambient audio.
  • Library puzzle: press the correct stone combination to reveal the Vault entrance.
  • Vault ending: enter the correct passphrase to win; otherwise… dust.
Built with just HTML, CSS, and tiny bits of JavaScript. No frameworks.
